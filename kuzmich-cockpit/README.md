# Единый пульт «Кузьмич» (Unitree G1)

Веб-пульт, который крутится на бортовом Jetson робота и собирает под одной крышей
всё управление: движение, руки, кисть, голову (эмоции + глаза), голос робота (TTS),
диалог с ИИ (локальная/облачная модель) и видео оператора.

Пульт **не переписывает** управление роботом — он мост к уже работающим сервисам V3:

| Подсистема | Интерфейс робота | Мостик |
|---|---|---|
| Движение | UDP `127.0.0.1:15100` `{vx,vy,wz}` | `backend/bridges/movement.py` |
| Руки | UDP `127.0.0.1:15001` `{action,seq}` | `backend/bridges/arms.py` |
| Кисть Inspire E2 | MODBUS TCP `192.168.123.210:6000` | `backend/bridges/hand.py` |
| Голова (LED + глаза) | ESP32-C3, WebSocket `ws://esp32-control.local:81/` | `backend/bridges/head.py` |
| Голос (TTS) | HTTP `127.0.0.1/api/audio/tts` | `backend/bridges/voice.py` |
| ИИ | Ollama `127.0.0.1:11434` | `backend/bridges/ai.py` |
| Видео | MJPEG `127.0.0.1:8091` | `backend/bridges/video.py` |

## Запуск (dev-машина, без робота)

```bash
python -m venv .venv
# Windows:  .venv\Scripts\activate
# Linux:    source .venv/bin/activate
pip install -r requirements.txt
python run.py
```

Открыть http://localhost:8080 — пульт работает в **dry-run**: команды не уходят на
робота, а логируются (видно в журнале внизу и в консоли). Так можно кликать всё
управление без железа.

## Запуск на борту робота

```bash
# на Jetson, где уже подняты сервисы V3 (motion receiver, arm-player, ollama, TTS, видео)
COCKPIT_DRY_RUN=0 python run.py
```

Затем открыть `http://<IP-робота>:8080` с ноутбука оператора в той же сети.

Адреса/порты и пресеты правятся через переменные окружения или в `backend/config.py`.

## Безопасность (interlock)

- **Ручной пульт** и **VR-телеоперация** — взаимоисключающие режимы. Пока активна
  телеоперация, ручные команды движения/рук/кисти блокируются (см. предупреждение в
  инструкции по телеоперации).
- **СТОП ⛔** — аварийная блокировка всей моторики: движение обнуляется, моторные
  команды не принимаются до сброса.
- Голова (LED/глаза), TTS и ИИ считаются безопасными и доступны всегда.
